# Sakaii OS Vision

## Positioning

Sakaii OS for Jellyfin is a full visual rethinking of Jellyfin built around a premium media operating system feel.

It is not intended to be a Netflix clone, a gaming UI, or a heavy neon theme. The intent is a quieter, more mature and more sculpted interface.

## Keywords

- premium
- minimal
- cinematic
- graphite
- glass
- precise
- calm
- immersive

## Desired Emotional Tone

- luxury without excess
- technical refinement
- comfort in low light
- clarity at a glance
- depth without clutter

## Visual Rules

- use layered surfaces rather than flat blocks
- keep blur soft and controlled
- prefer spacing and typography over loud effects
- keep the accent color reserved for actions and highlights
- avoid overusing glow, bloom, or saturated contrast

## Interaction Rules

- hover states should feel soft and intentional
- transitions should be subtle and slower than Nova UI
- motion should reinforce hierarchy, not call attention to itself
- JavaScript features should remain optional and lightweight

## Anti-Goals

- no aggressive neon aesthetic
- no oversized hover zooms
- no permanent pulsing UI
- no random color accents per component
- no dependency on visual gimmicks to feel premium
