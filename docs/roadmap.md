# Sakaii OS Roadmap

## Phase 1: Foundation

- define design tokens
- set the base shell colors and typography
- establish blur, border, and shadow rules
- add the main import structure

## Phase 2: Global UI

- redesign header
- redesign side navigation
- align buttons, inputs, chips, and menus
- refine overlays and modal surfaces

## Phase 3: Media Browsing

- redesign cards
- refine card hover behavior
- improve metadata readability
- polish home sections and rows

## Phase 4: Item Experience

- redesign item details page
- restyle action buttons
- improve logo, synopsis, and metadata layout
- refine cast and related sections

## Phase 5: Authentication

- redesign login screen
- define Sakaii background treatment
- refine field and CTA styling

## Phase 6: Playback

- subtitle styling cleanup
- progress bar and player control polish
- addon compatibility pass

## Phase 7: Optional JavaScript

- reintroduce only the features that improve comfort
- keep the DOM hooks resilient
- make optional features easy to disable
